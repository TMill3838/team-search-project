# 🛸 WARP GATE

## A keyboard shortcut search and filter engine with a retro aesthetic.

This app helps users find the best shortcuts for VS Code, Windows, and more. Built for CIS-376.

## 🛠️ What it Does
* **Search & Filter:** Find shortcuts by description or filter by the tool being used.
* **Smart Sorting:** Automatically sorts shortcuts by the keys used to keep everything organized.
* **Dynamic Data:** Uses the **JavaScript Fetch API** to retrieve shortcut data from a JSON source.
